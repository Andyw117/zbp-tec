## 🚀 Deployment Steps (GitHub Pages)

### Step 1 — Create a GitHub Repository

1. Go to [https://github.com](https://github.com) and log in.
2. Click **"New repository"** (top right "+" button).
3. Name it: `zhongbang-site` (or any name you like).
4. Set visibility to **Public** (required for free GitHub Pages).
5. Do NOT initialize with README (we'll upload our own files).
6. Click **"Create repository"**.

---

### Step 2 — Upload Files to GitHub

**Option A: Upload via Browser (easiest)**

1. On the new repository page, click **"uploading an existing file"**.
2. Drag and drop all three files:
   - `index.html`
   - `style.css`
   - `script.js`
3. Scroll down, write a commit message: `Initial site upload`
4. Click **"Commit changes"**.

**Option B: Using Git CLI**

```bash
# Clone the empty repo
git clone https://github.com/YOUR-USERNAME/zhongbang-site.git
cd zhongbang-site

# Copy the three files into this folder, then:
git add .
git commit -m "Initial site upload"
git push origin main
```

---

### Step 3 — Enable GitHub Pages

1. In your repository, click **"Settings"** (top tab).
2. In the left sidebar, click **"Pages"**.
3. Under **"Branch"**, select `main` and folder `/ (root)`.
4. Click **"Save"**.
5. Wait 1–3 minutes. GitHub will show:
   > "Your site is live at https://YOUR-USERNAME.github.io/zhongbang-site/"

---

### Step 4 — Access Your Live Site

Visit:
```
https://YOUR-USERNAME.github.io/zhongbang-site/
```

Replace `YOUR-USERNAME` with your actual GitHub username.

## 🔧 Customization Guide

### Changing Contact Info
Open `index.html` and search for the following to update:
- `+1 2137257531` — Phone number
- `Info@zbp-tec.cn` — Email address
- The address block in the `#about` and `#contact` sections

### Changing Colors
Open `style.css` and edit the `:root` variables at the top:
```css
:root {
  --c-teal:     #0a4a3f;   /* Dark teal (nav, cards) */
  --c-teal-lt:  #2d9c84;   /* Accent teal */
  --c-green:    #4cc98a;   /* Highlight green */
  --c-bg:       #f4f2ec;   /* Page background */
}
```

### Adding a Custom Domain
1. In GitHub Pages Settings, click **"Custom domain"**.
2. Enter your domain (e.g., `www.zbp-tec.cn`).
3. Add a CNAME DNS record at your domain registrar:
   ```
   Type: CNAME
   Name: www
   Value: YOUR-USERNAME.github.io
   ```

## 📌 Notes

- The contact form does **not** send emails by default (static site limitation).  
  To add real email functionality, consider integrating [Formspree](https://formspree.io) or [Netlify Forms](https://www.netlify.com/products/forms/).  
  Replace the `<form>` action with your Formspree endpoint:
  ```html
  <form action="https://formspree.io/f/YOUR-ID" method="POST">
  ```

- Google Fonts require internet access. For fully offline usage, download fonts and serve locally.
